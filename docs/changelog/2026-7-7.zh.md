## **2026-07-07**

**Futures V3 & Futures Testnet — 子账户划转更新**

- Futures V3 和 Futures Testnet（EN & ZH）更新 `POST /fapi/v3/subAccountTransfer`：
  - 鉴权参数说明表中将"主账户"统一更名为"母账户"。
  - 从参数表中移除 `signer` 字段（该端点不需要此参数）。
  - 新增**参数取值**表，明确各划转方向下 `user` / `fromAccountAddress` / `toAccountAddress` / `signature` 的对应填写规则：
    - 母转子、子转母（母账户签名）、子转子（母账户签名）
    - 子转母（子账户签名）、子转子（子账户签名）

**Futures V3 — 新增 Builder 查询接口**

- Futures V3（EN & ZH）新增两个 `USER_DATA` 接口：
  - `GET /fapi/v3/builder/userTrades`（权重 5）— 分页查询在当前调用者Builder代码下交易的用户成交历史，支持 `startTime` / `endTime` / `page` / `limit` 过滤；默认返回最近7天，最多可查30天。
  - `GET /fapi/v3/builder/approvedUserList`（权重 5）— 查询已将当前调用者授权为Builder的用户列表；支持可选 `startTime` 过滤；最多返回1000条记录。
