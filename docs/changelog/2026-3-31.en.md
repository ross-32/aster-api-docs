## **2026-03-31**

**Futures V3 API Updates**

- Added 4 new MMP (Market Maker Protection) endpoints to Futures V3:
  - `POST /fapi/v3/mmp` — Update User MMP: configure time window, frozen duration, qty/value/delta limits
  - `GET /fapi/v3/mmp` — Get User MMP: retrieve current MMP configuration by symbol
  - `DELETE /fapi/v3/mmp` — Delete User MMP configuration for a symbol
  - `POST /fapi/v3/mmpReset` — Reset User MMP for a symbol
