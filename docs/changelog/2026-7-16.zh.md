## **2026-07-16**

**Futures V3、Futures Testnet、Spot V3、Spot Testnet — Nonce 机制更新**

- 更新所有 V3 及 Testnet 基本信息页（EN & ZH）中的 **V3 Nonce 机制**说明：
  - Nonce 有效窗口从 **10 秒**扩展至服务端时间**前后 60 秒**。
  - 明确 nonce 状态（近期 nonce 列表及校验）维护在 **Agent 地址级别**，而非账户/用户级别——同一账户下的每个 Agent 地址独立维护、互不影响。
  - 将"每个用户仅维护最近 100 个 nonce"更新为"每个 Agent 地址仅维护最近 100 个 nonce"。

**Futures V3 — `GET /fapi/v3/builder/approvedUserList` 新增分页**

- Futures V3（EN & ZH）更新 `GET /fapi/v3/builder/approvedUserList`：
  - 新增请求参数：`endTime`（按授权时间上限过滤）、`page`（默认 `1`）、`limit`（默认 `100`，最大 `1000`）。
  - 响应结构从裸数组改为分页对象，包含 `rows[]`、`total`、`currentPage`、`totalPages`、`pageSize`、`hasMore`。每条记录字段（`userAddress`、`builderAddress`、`maxFeeRate`、`builderName`、`approveTime`）不变，现嵌套于 `rows[]` 下。
