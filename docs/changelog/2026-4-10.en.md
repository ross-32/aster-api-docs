## **2026-04-10**

**Futures V3 & Testnet API Updates**

- Added `POST /fapi/v3/sub-accounts/bind` — Bind Sub-Account endpoint to Futures V3 and Testnet (EN & ZH):
  - Requires two independent signatures: sub-account (`childSignature`) and master account (`signature`)
  - Master account signature must use the wallet private key (not the API Key)
  - Supports EVM (EIP-712) and Solana (Ed25519) address types
  - Only whitelisted addresses are supported; contact the project team for configuration
