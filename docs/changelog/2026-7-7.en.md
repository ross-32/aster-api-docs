## **2026-07-07**

**Futures V3 & Futures Testnet — Sub-Account Transfer Update**

- Updated `POST /fapi/v3/subAccountTransfer` in Futures V3 and Futures Testnet (EN & ZH):
  - Renamed "main account" to "master account" in the authentication parameters table.
  - Removed `signer` from the parameter table (it is not required for this endpoint).
  - Added **Parameter Values** table documenting `user` / `fromAccountAddress` / `toAccountAddress` / `signature` combinations for each transfer direction:
    - Master → Sub, Sub → Master (master-signed), Sub → Sub (master-signed)
    - Sub → Master (sub-signed), Sub → Sub (sub-signed)

**Futures V3 — New Builder Endpoints**

- Added two new `USER_DATA` endpoints to Futures V3 (EN & ZH):
  - `GET /fapi/v3/builder/userTrades` (weight 5) — Query paginated trade history of users trading under the caller's builder code. Supports `startTime` / `endTime` / `page` / `limit` filters; defaults to last 7 days; max lookback 30 days.
  - `GET /fapi/v3/builder/approvedUserList` (weight 5) — Query the list of users who have approved the caller's address as their builder; supports optional `startTime` filter; returns at most 1000 records.
